This is a hot folder where any cafr pdfs to be processed should be placed.

Files should be of the format: STATECODE_cafrYEAR.pdf
e.g. "AL_cafr2010.pdf"

Notes:
- Files starting with _ will be ignored
- Files that don't fit the above format will be moved to the "invalid" folder
- Files for states with no parser will be moved to the "invalid" folder
